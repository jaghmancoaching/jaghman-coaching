# Jaghman Coaching — دليل النشر 🚀

موقعك جاهز للنشر خلال **10 دقائق تقريباً**. اتبع الخطوات بالترتيب.

---

## المرحلة 1: التشغيل على جهازك (5 دقائق)

1. **ثبّت Node.js** (مرة واحدة فقط): حمّله من https://nodejs.org — اختر نسخة LTS.
2. افتح Terminal (أو CMD في ويندوز) داخل مجلد المشروع هذا، ثم نفّذ:

```bash
npm install
npm run dev
```

3. افتح المتصفح على `http://localhost:3000` — سترى الموقع كاملاً يعمل.

---

## المرحلة 2: النشر المجاني على Vercel (5 دقائق)

Vercel هي الشركة صاحبة Next.js — استضافة مجانية بشهادة SSL ورابط فوري.

### الطريقة الأسهل (بدون Git):
```bash
npm install -g vercel
vercel
```
- سجّل دخول بالبريد الإلكتروني عندما يطلب.
- اضغط Enter لقبول الإعدادات الافتراضية.
- خلال دقيقة يعطيك رابطاً مثل: `https://jaghman-coaching.vercel.app` ✅

### الطريقة الاحترافية (مع GitHub — تحديث تلقائي):
1. أنشئ مستودعاً على GitHub وارفع المجلد إليه.
2. ادخل https://vercel.com → **Add New Project** → اختر المستودع → **Deploy**.
3. كل تعديل ترفعه لاحقاً على GitHub يُنشر تلقائياً.

### ربط دومين خاص (اختياري):
اشترِ دومين مثل `jaghmancoaching.com` (من Namecheap مثلاً)، ثم في Vercel:
**Settings → Domains → Add** واتبع التعليمات — SSL مجاني تلقائياً.

## ⚠️ مهم: ما هو جاهز الآن وما يحتاج مرحلة ثانية

### ✅ جاهز للاستخدام الفعلي فوراً:
- الصفحة الرئيسية والهوية الكاملة (SEO مضبوط للأرشفة في جوجل)
- مكتبة البرامج المجانية بالجداول والشروحات الحركية
- زر واتساب المدرب (يعمل حقيقياً — يفتح محادثتك)
- نموذج "انضم كمدرب" وصفحات المدربين (كواجهة عرض)

### 🔧 يحتاج ربطاً ببنية خلفية (Backend) قبل استقبال أموال حقيقية:
| الميزة | الوضع الحالي | الحل للإنتاج |
|---|---|---|
| الدفع | تجريبي (Sandbox) | **Stripe** — يدعم هولندا وiDEAL |
| الحسابات وتسجيل الدخول | واجهة فقط | **Supabase Auth** (مجاني للبداية) |
| حفظ بيانات المشتركين والتقدم | في الذاكرة (يُمسح بالتحديث) | قاعدة **Supabase/PostgreSQL** |
| لوحة الإدارة | مفتوحة للعرض | حمايتها بدور "مسؤول" في Supabase |
| رسائل واتساب/البريد الآلية | واجهة محاكاة | WhatsApp Business API + Resend |

**نصيحة عملية:** انشر الموقع الآن كما هو كواجهة تسويقية — المكتبة المجانية تجلب الزوار من جوجل، وزر واتساب يحوّلهم إليك مباشرة لتدفعهم يدوياً (تحويل بنكي/Tikkie) حتى تجهز بوابة Stripe. هكذا تبدأ الربح من اليوم الأول بينما تُبنى المرحلة الثانية.
